# RQ3：泛化性与可扩展性

## 实验问题

> MASDiff 能否泛化到标准多智能体基准（稀疏奖励），并随智能体数量的增加有效扩展？

详细实验结果见 [MPE 场景文档](../scenarios/mpe.md)。

## 核心结论

**泛化性**：MASDiff 无需任何场景特定设计，直接应用于 Grid Navigation 和 Push Box 两个稀疏奖励任务，均优于 QMIX、EITI/EDTI、EMC 等专用 MARL 方法。

**可扩展性**：从 10 个智能体扩展至 1000 个（Grid Nav），计算时间增长平稳，且始终低于对比方法。在 Push Box 的 100 智能体规模下成功率达到 1.0。

**计算效率**：独立离线策略训练天然支持并行化，是 MASDiff 高效扩展的根本原因。
