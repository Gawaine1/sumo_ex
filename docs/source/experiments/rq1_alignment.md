# RQ1：宏观目标对齐

## 实验问题

> MASDiff 能否有效最小化宏观差距 $\sum_{t=1}^T \|X_t - X_t^*\|$，实现目标集体行为？

## 实验设置

- **场景**：重庆某区 SUMO 交通网络（114 条路段，45 个交叉口，1557 辆车）
- **目标**：单目标 LCC 对齐（最大连通高速路段）
- **对比基线**：CGAN、LDM_IMIT、LDM_CFG、LDM_PCFG

## 结果

MASDiff 在所有对比方法中收敛到最小宏观差距，且收敛速度最快、曲线最稳定。

```{admonition} LDM_CFG 和 LDM_PCFG 失败的原因
:class: warning
将宏观差距直接作为条件信号输入扩散模型，提供的探索引导不足。进化搜索在轨迹空间中比条件生成提供了更有效的全局探索。
```

```{admonition} LDM_IMIT 失败的原因
:class: warning
纯模仿学习从生成轨迹中提取策略是不够的，离线 RL 对于提取鲁棒的奖励最大化策略至关重要。
```

## 逐分钟 LCC 对比

MASDiff 生成的集体交通模式与真实世界的 LCC 时序高度吻合，证明了成功的宏观对齐。这说明通过进化扩散搜索找到的轨迹-奖励对，能够引导智能体产生与真实交通流相似的涌现行为。
