# 简介

## 背景与动机

在去中心化多智能体系统中，同时实现以下两个目标是一项根本性挑战：

1. **集体行为学习**：学习分布式策略，使智能体的集体行为与给定的宏观目标模式对齐。例如，给定真实交通拥堵数据，能否学到每辆车的导航策略，使仿真拥堵与观测数据吻合？

2. **反事实推理**：利用学到的奖励模型预测新条件下的系统输出。例如，若某路口禁止左转，拥堵格局将如何变化？

现有方法的局限性：

| 方法类别 | 代表方法 | 局限性 |
|---------|---------|--------|
| 行为克隆（BC） | Zhou et al., 2024 | 需要大量专家轨迹；不学奖励函数 |
| 逆强化学习（IRL） | Wang et al., 2024 | 需要专家示范；难以扩展到千级智能体 |
| RLHF | Casper et al., 2023 | 需要人类反馈；无法学个体奖励 |
| 内在奖励 MARL | QMIX, MAVEN, EMC | 只学隐式奖励；无法做反事实推理 |
| 种群 MARL | PBT, self-play | 无法学个体奖励函数 |

MASDiff 是第一个无需专家示范、同时解决以上两个目标的框架。

## 核心思想：轨迹到策略

MASDiff 提出了全新的 **轨迹到策略（Trajectory-to-Policy）** 视角：

```{mermaid}
graph LR
    A[随机策略<br/>rollout] -->|探索轨迹空间| B[联合轨迹 τ]
    B -->|条件扩散模型| C[奖励序列 R]
    C -->|离线 RL| D[改进策略 π]
    D -->|宏观评估| E{宏观差距 D}
    E -->|进化选择 + 变异| B
    style E fill:#e74c3c,color:#fff
    style C fill:#2980b9,color:#fff
```

传统方法在非平稳环境中同时做探索与利用，两者相互干扰导致不稳定。MASDiff 将它们**彻底分离**：

- **探索**在轨迹空间通过进化搜索完成，直接以宏观目标为适应度函数。
- **利用**通过稳定的离线 RL 从选定轨迹中提取策略。
- **条件扩散模型**作为桥梁，为每条轨迹匹配最优奖励，并通过截断扩散实现邻域变异。

## 四大核心贡献

**① 轨迹到策略学习**
将探索与利用解耦，避免非平稳性冲突，实现稳定的策略学习。

**② 进化扩散自举**
进化扩散机制生成轨迹-奖励对，无需专家示范即可学习显式奖励模型。

**③ 目标导向协调**
宏观目标直接作为进化适应度函数，引导搜索朝向期望的涌现行为。

**④ 可扩展架构**
独立的离线策略学习与精英保留机制支持并行训练，可扩展至数千智能体。
